# 暗幕排版台 · Dark Type Studio

一个用于制作黑底红白大字封面与视频开头标题页的纯前端排版工具。打开网页即可修改文字、拖动图层、上传本地字体，并导出高清 PNG、JPG 或 WebP。

## 当前版本

`v1.1.1`

当前内置两套模板：

- 主页方形封面：1080 × 1080
- 视频开头标题页：1920 × 1080

## 已实现功能

- 模板切换与一键恢复
- 文字、印章、装饰线图层
- 画布拖动定位与方向键微调
- 字体、字重、字号、颜色、字间距、行距、宽度、对齐方式
- 描边、阴影、透明度
- 本地 TTF、OTF、WOFF、WOFF2 字体临时加载
- 新增、复制、删除、排序、隐藏图层
- 画布背景色修改
- 1×、2×、4× PNG、JPG、WebP 导出
- 1:1、3:4、4:3、4:5、9:16、16:9 和 1.91:1 社媒画布预设
- 拖拽到画布中心时的横纵参考线、吸附与可用设备的触觉反馈
- 浏览器本地保存与读取
- 项目 JSON 导入和导出
- 撤销与重做

## 本地运行

这是一个无构建步骤的静态网页项目。任选一种方式启动：

```bash
python -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```

也可以使用 VS Code 的 Live Server，或直接部署到 GitHub Pages、Vercel、Netlify。

## GitHub Pages

推送 `main` 后，GitHub Actions 会运行检查并将精简的静态文件部署到 GitHub Pages。

## 使用提示

1. 点击模板进入对应画布。
2. 点击画布元素或右侧图层列表选择图层。
3. 在右侧属性栏修改内容与排版。
4. 拖动元素调整位置；方向键移动 1 像素，Shift + 方向键移动 10 像素。
5. 选择导出倍率并导出图片。

本地上传字体不会自动写入项目仓库。重新打开页面后，需要重新上传对应字体文件；项目 JSON 会保留字体名称，便于恢复。
