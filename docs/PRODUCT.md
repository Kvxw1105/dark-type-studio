# 暗幕排版台 v0.1 产品冻结说明

## 产品目标

将高辨识度的黑底红白字封面与视频开场标题页沉淀为可反复复用的排版工具。用户能够在不依赖设计软件和 AI 生图的情况下，稳定获得文字准确、尺寸可控、可继续编辑的视觉成品。

## v0.1 范围

### 模板

1. 主页方形封面，1080 × 1080。
2. 视频开头标题页，1920 × 1080。

### 编辑能力

- 修改文字内容。
- 拖动图层并输入精确坐标。
- 修改字体、字重、字号、颜色、字间距、行距和最大宽度。
- 修改描边、阴影、透明度。
- 上传本地字体。
- 新增、复制、隐藏、删除和调整图层顺序。
- 修改背景颜色。

### 导出能力

- PNG、JPG、WebP。
- 1×、2×、4× 导出倍率。
- 项目 JSON 导入与导出。
- 浏览器本地保存。

## 技术路线

- 原生 HTML、CSS、JavaScript。
- Canvas 2D 负责实时预览与导出。
- FontFace API 负责本地字体加载。
- localStorage 负责轻量项目保存。
- 不依赖后端，不依赖构建工具，适合直接部署 GitHub Pages。

## 暂不进入 v0.1

- 用户账户与云端同步。
- 多人协作。
- AI 自动生成内容。
- 视频编码与动画时间轴。
- 完整矢量绘图系统。
